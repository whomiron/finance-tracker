# Finance Tracker — система учета личных финансов

Finance Tracker — это веб-приложение, позволяющее пользователю фиксировать расходы, управлять счетами, отслеживать выполнение бюджетов и контролировать достижение финансовых целей.

---

## Функционал

### Расходы
- Добавление новых расходов  
- Удаление и редактирование операций  
- Категоризация расходов  
- Привязка расходов к конкретным счетам  
- Просмотр операций за выбранный период

### Бюджеты
- Создание бюджетов по категориям  
- Автоматический подсчёт потраченной суммы  
- Расчет остатка бюджета  

### Счета
- Автоматическое создание основного счёта при регистрации  
- Хранение баланса и информации о счёте  
- Просмотр связанных транзакций

### Рекомендации по расходам
- Анализ динамики трат  
- Подсказки по оптимизации расходов в выбранных категориях  

### Финансовые цели
- Настройка целей и целевых сумм  
- Отслеживание прогресса достижения  
- Отображение активных целей пользователя

### Пользователи и безопасность
- Регистрация и авторизация  
- Защита маршрутов с помощью Spring Security

## Запуск проекта

Требование: установленный Docker.

```bash
docker compose up --build
```

## После успешного запуска приложение будет доступно по адресу:
## http://localhost:8080


## Приложение разворачивается на двух контейнерах: Spring Boot и PostgreSQL. База данных будет автоматически создана при запуске.
